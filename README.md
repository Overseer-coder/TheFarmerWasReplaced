# TheFarmerWasReplaced
The Farmer Was Replaced é um idle game de programação onde automatizamos drones para gerenciarem uma fazenda, plantando e colhendo. Há desafios extras como a resolução de labirintos e de um desafio como o antigo "Snake"(popular "Jogo da Cobrinha"). Este repositório armazena os scripts utilizados durante o jogo.
